- **파일**: [`index.html`](index.html)
- **바로 실행 (배포된 사이트)**: https://lkw07309-gif.github.io/lkw/

## 2. 📊 발표 자료 (PPT)

PowerPoint 발표 자료를 텍스트·영상 손실 없이 원본 그대로 보관했습니다. 슬라이드 8장, 이미지 8개, 영상 2개가 모두 포함됩니다.

- **폴더**: [`presentation/`](presentation/)
- **원본 파일**: [`presentation/presentation.pptx`](presentation/presentation.pptx) (다운로드 후 PowerPoint에서 열기)
- **웹 슬라이드쇼로 바로 보기 (배포된 사이트)**: https://lkw07309-gif.github.io/lkw/presentation/
  - 슬라이드 1~6은 이미지, **7·8번은 영상**이 재생됩니다.

자세한 내용은 [`presentation/README.md`](presentation/README.md) 참고.
