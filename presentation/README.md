# 발표 자료 (Presentation)

PowerPoint 발표 자료를 텍스트·영상 손실 없이 원본 그대로 보관한 폴더입니다.

## 파일 구성

- **`presentation.pptx`** — 원본 PowerPoint 파일 (완전 보존, MD5 무결성 확인됨)
  - 슬라이드 8장, 이미지 8개, 영상 2개 모두 포함
  - PowerPoint / 구글 슬라이드 등에서 열면 원래대로 재생됩니다
- **`media/`** — 원본에서 추출한 미디어 (PowerPoint 없이 바로 보기/다운로드용)
  - `image1.png` ~ `image6.png` : 슬라이드 1~6 이미지
  - `image7.jpeg`, `image8.jpeg` : 슬라이드 7~8 이미지
  - `VAHOSK5qjvs.mp4` : 슬라이드 7 삽입 영상
  - `VAHOSGxEj_0.mp4` : 슬라이드 8 삽입 영상

## 참고

- 영상은 슬라이드 7·8에 삽입되어 있습니다.
- GitHub은 `.pptx` 미리보기를 지원하지 않으므로, 웹에서 바로 내용을 보려면 `media/` 폴더의 이미지·영상을 이용하세요.
- 원본 그대로 필요하면 `presentation.pptx`를 다운로드해 PowerPoint에서 열면 됩니다.
